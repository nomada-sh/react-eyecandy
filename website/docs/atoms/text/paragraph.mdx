# Paragraph

import { Typography } from 'antd';

export const Paragraph = Typography.Paragraph;

Examples:

```jsx
<Paragraph>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit.
    Proin malesuada vulputate purus.
    Sed vitae vulputate diam, a hendrerit felis.
    Fusce auctor laoreet scelerisque. Nullam sed mollis nisl.
    Fusce vitae consequat tortor. Fusce congue metus vel est efficitur, in tincidunt tortor facilisis.
    Duis euismod finibus velit et egestas.
    Sed justo sapien, porttitor eget semper nec, ullamcorper vel erat.
    Sed accumsan erat ut lectus condimentum, ut rutrum nunc facilisis.
    Quisque lacinia non lectus fermentum dapibus.
    Praesent eget justo et quam interdum accumsan.
    Quisque mauris urna, mattis eu pellentesque venenatis, cursus eu augue.
    Etiam sodales, turpis dignissim elementum ultricies, justo tellus placerat leo, ac consequat turpis augue vel justo.
</Paragraph>
```
